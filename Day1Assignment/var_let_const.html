Var let Const

1.var

var is globally and locally scoped. If var is used outside the function,it can be used for the whole window. If var is used inside the function, it can be accesed only inside the function.

var variable can redeclared and updated and does not show error

2.let

let is block scoped

let can be updated and not redeclared

3.const

variable declared with the 'const' maintains the constant values.

const cannot be re declared or updated.

Every const variable must be initialized at the time of declaration.
