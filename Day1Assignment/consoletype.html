Console Methods

Console objects which provides the user access to the browser for the debugging purposes.

1.log()

It is used to print the output to the console.Inside teh log may be strinf , boolean , integer, object,array.

2.error()

It is used to log error message to the console.In default, it is highlight in red color.

3.warn()

It is used to log warn message to the console.In default, it is highlight in yellow color.

4.clear()

It is used to clear the console. There willbe text 'Console was cleared' after using this method.

5.time() and timeEnd

It is used to calculate the time of the operation performed. It is in ms.

6.table()

It is used to generate table in the console. The input may be array or object.

7.count()

It is used to count the number that the functin hits the count method.

8.group() and groupEnd()

It is used to group the contents in the separate block.groupEnd is used to end the block.

9.custom console log

It is used to add css styles to the text in the console.
